enlace para entra a la herramienta de administracion del sitio WSAT:

Para levantar el sitio con cmd de admin:

iisexpress.exe /path:C:\Windows\Microsoft.NET\Framework\v4.0.30319\ASP.NETWebAdminFiles /vpath:"/ASP.NETWebAdminFiles" /port:8082 /clr:4.0 /ntlm

Para acceder al sitio:
http://localhost:8082/asp.netwebadminfiles/default.aspx?applicationPhysicalPath=C:\Users\user\Documents\app\prueba36\Main\prueba36web\prueba36web\&applicationUrl=/
